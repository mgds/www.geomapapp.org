--- layout: default title: GeoMapApp Chromebook Installation ---

GeoMapApp is a Java desktop applicaton and it must be downloaded to your computer. This is not a default feature on Chromebooks, but there is a way. If you agree to the Terms of Use, follow these instructions.

For your first launch, you will first have to enable the Linux virtual machine and download Java:

  1. On the Chromebook, open the launcher (circle menu) and click Settings.
  2. On the left sidebar, click "Advanced", then "Developers". You may have to scroll.
  3. Where it says "Linux development environment", click the button that says "Turn on", then click "Next". ChromeOS may need to update to enable this.
  4. Set the username and disk size you want (the default options will work) and click "Install". Wait for the installation to complete.
  5. When the installation completes, a Terminal window will open. Optionally, right click (or the equivalent) on the Terminal icon in the shelf and select "Pin" for easy access in the future. You will need to open a Terminal window to run GeoMapApp in the Linux environment.
  6. In the terminal window, type the following and press "Enter"/"Return" to install Java: sudo apt-get install default-jdk (click to copy to clipboard)
  7. When asked if you want to continue using more disk space, agree by typing "y" (not case-sensitive) and press "Enter"/"Return"
  8. Confirm that the installation worked by typing the following into your Terminal window, which should tell you which version of Java you installed, then pressing "Enter"/"Return": java -version (click to copy to clipboard)
  9. Download the executable JAR from the GeoMapApp Linux download page.
  10. When the download finishes, click the "Show in folder" button.
  11. Click and drag GeoMapApp.jar from that folder into where it says "Linux files" on the left side. This will copy it into the Linux development environment.
  12. Back in the Terminal window, launch GeoMapApp by typing the following command, followed by the "Enter" or "Return" key (may take several seconds): java -jar GeoMapApp.jar (click to copy to clipboard)

For subsequent launches, you will naturally have fewer steps to follow:

  1. Open the Terminal, either from the launcher or the shelf.
  2. Under Linux, click where it says "penguin", which is the name of the Linux development environment you enabled earlier. This will open the Terminal window.
  3. Type the following command in the Terminal window, then press "Enter"/"Return": java -jar GeoMapApp.jar (click to copy to clipboard)
  4. Wait for GeoMapApp to launch; this may take several seconds.
