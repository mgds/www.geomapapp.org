Java Web Start (also known as JavaWS or javaws) is a framework developed by Sun Microsystems that allows users to start application software for the Java Platform directly from the Internet using a web browser.

If you are experiencing issues with the Web Start launch read the following walkthrough.

  • Jump to webstart Walkthrough for Windows.


    • GeoMapApp Web Start: Mac OS

    Step 1:
    Launch GeoMapApp from the link on our main webpage. Step 2:
    A dialog window should prompt you to either open or save a VirtualOcean.jnlp file. You must choose to open with 'Java Web Start', if you do not see this as the default option then proceed to Step 3. Otherwise if the option is present press ok and proceed to Step 7. Step 3:
    Choose the 'Open with' option. Choose the 'Other' option and open with TextEdit.app. Step 4:
    Open the Finder and go to Applications -> Utilities -> Java Preferences.app to launch the Java Preferences. Step 5:
    Under the Network Tab click on View Cache Files Step 6:
    A new window should appear. Choose Show Applications from the top dropdown selection. Next click the GeoMapApp Icon then hit the green play button icon located on the top menu bar. Step 7:
    Java Web Start is now running and launching the GeoMapApp application. Step 8:
    You should allow both security cerficate prompts and GeoMapApp will start.

    If you are still experiencing problems:

    1. Save this jnlp file onto you desktop from Step 2.
    2. Open the Terminal.app located in Applications -> Utilities -> Terminal.app.
    3. Type the command below and press return.
      cd Desktop
    4. Type the next command below and press return
      javaws VirtualOcean.jnlp
    5. The webstart should now begin. Proceed to Step 6 and Step 7.



    GeoMapApp Web Start: Windows 7

    Step 1:
    Launch GeoMapApp from the link on our main webpage. Step 2:
    Java Web Start should automatically run and launch the GeoMapApp application. Step 3:
    You should allow both security cerficate prompts and GeoMapApp will start.

    If you are still experiencing problems:

    1. Open the Java Control Panel by going to Control Panel -> Programs -> Java.
    2. The Java Control Panael should have opened. Under the General tab in the Temporary Internet Files section click on 'View'.
    3. A new window should appear. Choose Show Applications from the top dropdown selection. Next click the GeoMapApp Icon then hit the green play button icon located on the top menu bar.
