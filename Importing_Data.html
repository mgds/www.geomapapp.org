Accepted formats:

  • Microsoft Excel Tables
  • Ascii Tables:
    • One row per line of ascii text
    • Columns must be tab-separated
  • Either of the above, copied to the clipboard

Requirements:

  • The first row in the table must be column headings.
  • There must be latitude and longitude columns (with any heading allowed)

Data may be imported from local files or remote URLs.
