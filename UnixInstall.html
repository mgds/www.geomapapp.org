    • Minimum System Requirements:

    • Java 8 (Java may be downloaded from here)

      • NOTE: The default GNU Java compiler installed on some Linux platforms does not include all the libraries needed by GeoMapApp. Install Java as above.

    • 60 MB of free disk space.
    • 512 MB of memory.
    • An internet connection for access to data.

    To start the program, type the following in a command window:

    % cd <directory with GeoMapApp.jar>
    % java -Xmx512m -jar GeoMapApp.jar
