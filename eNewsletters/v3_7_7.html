Dear GeoMapApp community,

We are pleased to announce a new release of GeoMapApp, version 3.7.7.

This release includes enhanced functionality and several bug fixes.

Shapefile import
Shapefiles in geographic, UTM and other projections may now be imported.

GeoTIFF grid import
GeoTIFF grids in World Mercator projection are now correctly identified. An issue detecting No-Data values has been fixed.

GeoTIFF images import (Beta release)
Functionality for importing unrotated GeoTIFF images (via File > Add Image Overlay) has been enhanced to automatically detect bounding coordinates.

Import tables and spreadsheets
Enhanced capability to identify the latitude and longitude columns from the column headers.

Digital Seismic Reflection Profiles portal
Added a "Cruise Info" button linked to cruise-level metadata.

Multibeam Swath Bathymetry portal
Selecting a track line has been made easier.

Default Memory Size
To reflect the larger memory size of modern laptop and desktop computers, the default memory allocation has been increased for better handling of large data sets.

Radar Profiles of Antarctic Ice Thickness portal
Improvements to the portal for the Antarctica Gamburtsev Province (AGAP) and Lake Vostok radar data surveys.

Coordinate Range
Switching from the default ±180 degrees longitude range to 0-360 degrees now works correctly.

Data set units
The Z units are now correctly identified for built-in gridded data sets.

Pop-up windows
Improved the default locations of these windows.

More details about the release may be found here.
