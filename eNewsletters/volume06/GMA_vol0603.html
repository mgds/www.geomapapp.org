Dear GeoMapApp community,

We are pleased to announce a new release, version 3.3.0.

Highlights of the release include an update to the Petrological Database of the Ocean Floor (PetDB) geochemistry portal, provision to save base map grids at greater resolution, and a choice of different symbols for plotting data.

The PetDB portal now includes data samples through March 2012 as well as simplified controls for data selection.

A choice of symbol is now available for plotting point data. Location information in spreadsheets and tabular data sets can be plotted as circles, squares, triangles or stars. With multiple data sets the symbol used for each data set can be colored and scaled independently. For example, the location of different types of instruments can be plotted with different symbols. The choice of plot symbols is available as a Configure option for both built-in data and for imported spreadsheets and tabular data sets.

Another new feature is that when viewing the GeoMapApp base map - the Global Multi-Resolution Topography (GMRT) synthesis - or manipulating the GMRT grid, there is now an option to save the underlying grid with higher resolution than that shown on-screen. The option is accessed via the Save Map menu selection.

Several bugs reported by the GeoMapApp community have also been addressed. Please check the GeoMapApp development history page for more details.
