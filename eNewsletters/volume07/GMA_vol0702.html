Dear GeoMapApp community,

We are pleased to announce a new release, version 3.3.4.

The GeoMapApp community has reported compatibility issues with Microsoft Excel 2011 (Mac). Users where unable to import their spreadsheets into the application. This issue has been addressed in this release. Please check the GeoMapApp development history page for more details.
