Dear GeoMapApp community,

We are pleased to announce a new release, version 3.3.8.

Highlights of the release include updates to the icons on the main tool bar in GeoMapApp. Also, a new Projection Menu (still in beta development) has been added to the main GeoMapApp window. It is located between the Windows and Help Menus. The new menu allows users to switch projection without restarting the application. If a user is on one projection they have the option to switch to one of the other two projections.

We've also enhanced the Excel Spreadsheet import function. Excel spreadsheets (.xls / .xlsx) with simple addition, subtraction, division and multiplication formulas in the cells can now be imported in GeoMapApp.

The File>Import> Web Map Service (WMS) has been upgraded to handle version 1.3.0. Appending 1.1.1 or 1.3.0 to the WMS GetCapabilities URL specifies the WMS version. If no version is given, the remote WMS server sets a default.

Several improvements to the Profile Tool have been made. The window layout is more robust; hemisphere menus for start/end points are now hidden until the user specifies the Degrees and Decimal Minutes format; and more.

This release includes several other enhancements and bug fixes reported by the GeoMapApp community which have also been addressed. For a full list of changes take a look on our GeoMapApp development history page for more details.

As always, you may email us with any problems, comments, and/or suggestions to info@geomapapp.org
