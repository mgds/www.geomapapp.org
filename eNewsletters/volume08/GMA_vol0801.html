Dear GeoMapApp community,

We are pleased to announce a new release, version 3.3.9.

Highlights of the release include several enhancements and bug fixes. The Projection Menu located between the 'Windows' and 'Help' menus now works properly, allowing users to change the map projection without restarting the application. If viewing the Mercator, South Polar or North Polar map, a user has the option to switch to one of the other two projections.

The importing of grids in grd98 format for big-endian binary files has been fixed. This option can be found on the File Menu> Import a 2-D Grid File> From Local File System Grid File. A drop down list of different grid formats to import will appear, including the GRD98 format. When importing the grid file, the information button on the top right has always given extra header information to the user. Now, it also alerts users if an imported grid file has no readable header information. User instructions for importing Shapefiles, 2-D Grids, Tables and Spreadsheets have been updated. See the Import Data Format Requirements page, located under File > file type.

Improvements to the Multibeam Swath Bathymetry portal now update the cache automatically when there is a new version available. Users can still manually refresh, clear or disable the cache via the Preferences section found under the File menu. The Multibeam Swath Bathymetry Portal can be found in the 'Portals' section of the menu.

Other improvements have been made to the Lasso Tool for all tabular data and for use within the PetDB portal. The Earthquake Focal Mechanism portal has been updated to include data from May 2012 to August 2013. The Ocean Drilling portal no longer causes errors or freezes the portal when a drill site is selected. Also, the zoom-to-region functionality now works correctly when selecting certain menu items such as Regional Maps, Geological Maps and more.

This release includes several other enhancements and bug fixes requested by the GeoMapApp community. Visit the GeoMapApp development history page for a complete list of changes and more details.

As always, you may email us with any problems, comments, and/or suggestions to info@geomapapp.org
