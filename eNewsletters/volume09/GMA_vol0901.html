Dear GeoMapApp community,

We are pleased to announce a new release, version 3.5.0.

Highlights of this release include a major upgrade in GeoMapApp to Java 7. GeoMapApp will no longer work with older versions of Java. Please ensure that you have Java 7 or higher installed on your machine.

For Mac users, GeoMapApp now supports the new Gatekeeper security in Mac OS X Lion v10.7.5 and higher. GeoMapApp also has a new installer and is now signed with a trusted Apple Developer ID which should address Mac downloading issues. However, due to changes in Mac OS X bundling, users with Mac OS X Snow Leopard v10.6 and below will need to continue using GeoMapApp version 3.4.1, available here. Our FAQ page provides more information.

The GMRT base map images now load faster in all projections allowing the map to be refreshed more quickly when zooming and panning. Additionally, the underlying GMRT grids are provided at greater precision.

An Undo Last Zoom action is now available. The new feature is found in the toolbar next to the other zoom icons, and will undo the most recent zoom action once.

A new Web Feature Service for the IEDA EarthChem database provides access to geochemical data and samples information.

Other improvements and bug fixes requested by the GeoMapApp user community have also been included. Examples are found in the profile tool that now always has two or more annotation labels in the y-axis; and, when importing tabular data files, GeoMapApp now treats lines that begin with a '#' symbol as a comment. Visit the GeoMapApp development history page for more details and a complete list of changes.

As always, please email us with any problems, comments, or suggestions at info@geomapapp.org or make a post to the GeoMapApp User Forum.
