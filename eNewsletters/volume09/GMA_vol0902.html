Dear GeoMapApp community,

We are pleased to announce a new release, version 3.5.2, to accommodate the pending expiration of the Verisign security certificate. A full semi-annual release of GeoMapApp will be made in the late fall.

This release includes some other updates and bug fixes.

The Grid Import navigation window for Mac users has been fixed, as have zoom issues encountered when loading some grids. User prompts have been clarified for Web Feature Services.

The Earthquake Focal Mechanism Solutions Portal has been updated with Global Centroid Moment Tensor Project final CMT solutions for January 1976 to April 2015.

Visit the GeoMapApp development history page for more details and a complete list of changes.

As always, please email us with any problems, comments, or suggestions at info@geomapapp.org or make a post to the GeoMapApp User Forum.
