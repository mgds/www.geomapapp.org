Dear GeoMapApp community,

We are pleased to announce a new release, version 3.6.11.

This release includes several bug fixes and a data portal content update.

Portal Update: The geochemistry PetDB portal is updated to February 2021 to include analytical and compiled geochemical data derived from more than 76,500 mainly seafloor samples currently in the PetDB database. The geochemical data can be filtered, displayed, and saved in a variety of ways.

Bug Fixes: The following issues were amongst those that have been fixed in this new version:

  • PetDB portal external URLs.
  • Ocean Drilling portal external URLs.
  • Grid import: ESRI ASCII and Polar ASCII grids now import properly. Number of grid nodes now displays correctly. Upside-down grids can now be flipped the right way up.
  • Save Session layer ordering and contour bolding.
  • Multibeam swath bathymetry portal caching and load times.
  • Issues with the Distance Scale and Country Boundaries overlays.
  • Analog Seismic Profile portal: Problems with imported digitized files.
  • Island land elevations at high zoom factors.
  • Other bug fixes necessary to maintain GeoMapApp have also been implemented.

As always, you may email us with any problems, comments, or suggestions at info@geomapapp.org
