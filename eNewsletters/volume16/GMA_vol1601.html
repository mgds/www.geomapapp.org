Dear GeoMapApp community,

We are pleased to announce a new release of GeoMapApp, version 3.7.0.

This release includes several bug fixes, updates to two data portals, and improvements to the Digitizer function and to the tabular data capabilities.

The PetDB portal has been improved to better handle the quarterly PetDB updates. However, to continue using the PetDB portal, users will need to upgrade to this new version of GeoMapApp.

The Waypoints and Survey Planner Portal and the toolbar Digitizer function have both been enhanced to allow intermediate points to be added to an already-digitized segment.

Some bugs in the tabular data capability have been fixed: By default, imported tabular data points are plotted as circles ("Stations") on the map. The Configure button allows the user to switch to "Track" mode which joins the points with a continuous line. These lines are now displayed properly. Also, imported text files containing ">" segment separators are now correctly handled.

This release also includes bug fixes necessary to maintain GeoMapApp.

You may email us with any problems, comments, or suggestions at info@marine-geo.org.
