Dear GeoMapApp community,

We are pleased to announce a new release of GeoMapApp, version 3.7.4.

For this release only, this new version must be downloaded manually.

Mac download: https://www.geomapapp.org/MacInstall.html

Windows download: https://www.geomapapp.org/MSInstall.html

JAR download: https://www.geomapapp.org/UnixInstall.html

The version 3.7.4 release includes several bug fixes:

  • The Save Session functionality now correctly captures the data point shape, size and color when multiple tabular data sets are displayed in GeoMapApp.
  • A fix for exported KMZ images.
  • A fix to the GMRT base map grid tiling display.
  • Other minor improvements and bug fixes.

In addition to these updates, we would like to inform users that since October 2023 GeoMapApp has been running from the Cloud. This has yielded improved stability and faster performance.

More details about the release may be found here.

As always, you may e-mail us with any problems, comments, or suggestions at info@marine-geo.org.
