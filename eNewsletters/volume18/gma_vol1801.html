Dear GeoMapApp community,

We are pleased to announce a new release of GeoMapApp, version 3.7.5.

This release includes new functionality and several bug fixes.

Import grids

GeoMapApp is now able to import GeoTIFF-formatted grids. The grids can be in either geographic or UTM projection. This addition of GeoTIFF grid import capability complements the existing functionality for importing netCDF and ESRI ASCII/Binary grids.

Multibeam Swath Bathymetry portal

This portal allows users to quickly identify and download multibeam swath data files. The portal now shows when the swath file content was last updated.

PetDB portal

This analytical geochemistry portal now shows when the PetDB data content was last updated.
