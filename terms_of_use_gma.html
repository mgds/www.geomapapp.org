To Cite GeoMapApp

In the text and figure captions, please reference the use of GeoMapApp and include the URL https://www.geomapapp.org. See example below.

In addition, if you have generated a map or image using the default base map which is called the Global Multi-Resolution Topography (GMRT) synthesis, please also include the Ryan et al., 2009 citation and include it in your reference list. See example below.

Citation examples:

(a) For an image created with GeoMapApp this citation would be appropriate:
  Figure made with GeoMapApp (www.geomapapp.org) / CC BY.

(b) If your image also uses the GMRT base map (the default base map in GeoMapApp) the following would be an appropriate citation:
  Figure made with GeoMapApp (www.geomapapp.org) / CC BY / CC BY (Ryan et al., 2009).

And, the full citation for the Ryan et al., 2009 paper to be included in your reference list is as follows:

  • Ryan, W. B. F., S.M. Carbotte, J. Coplan, S. O'Hara, A. Melkonian, R. Arko, R.A. Weissel, V. Ferrini, A. Goodwillie, F. Nitsche, J. Bonczkowski, and R. Zemsky (2009), Global Multi-Resolution Topography (GMRT) synthesis data set, Geochem. Geophys. Geosyst., 10, Q03014, doi:10.1029/2008GC002332.
  • Data doi: 10.1594/IEDA.0001000

Accuracy

We make no claims to accuracy of the data and information available through GeoMapApp. We ask that you notify us immediately of any errors that you discover, and we will make every effort to correct them. We encourage contributions of data for inclusion in GeoMapApp.

Any data in GeoMapApp should not be used for navigation purposes.
